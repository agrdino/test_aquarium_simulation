# Bài test cho Ứng viên Unity: Middle Level
### Ghi chú:
- git: Hãy fork project sau về và bắt đầu. Gửi link git của bạn khi hoàn thành.
- Phiên bản Unity: 6.0
- Bạn được phép sử dụng Assets bên ngoài, miễn là nó phù hợp với project.
- Bạn được tự do chỉnh sửa project.
- Hãy đảm bảo project hoạt động tốt khi nộp bài.
- Hãy gửi kèm 1 video ngắn demo kết quả của bạn nhé.

## Context:
- Đây là Game 2D giả lập Aquarium Simulation.
- Support các thiết bị Mobile cấu hình yếu.
- Hiển thị màn hình dọc.
- Target 60 FPS.
- Art Style đơn giản, phù hợp cho mọi giới tính, lứa tuổi.
- Project cung cấp sẵn các hình ảnh cơ bản.

## Yêu cầu của bài test:
- Start Game với 300$ và hồ cá có sẵn 3 con cá bơi lội bên trong 
- Hiển thị 1 nút bấm Purchase Random Fish: giá mỗi con cá là 10$. Mua thành công thì xuất hiện trong hồ.
